# Projeto de Machine Learning - Classificação do Dataset Iris

## Sobre o Projeto
Este projeto tem como objetivo implementar um pipeline completo de Machine Learning, desde o carregamento e análise de dados até a construção, avaliação e validação de modelos preditivos.
O problema abordado é a classificação de flores da base Iris, um dos datasets mais clássicos da área, frequentemente utilizado como "Hello World" em Machine Learning.

## Objetivos
- Compreender o fluxo completo de um projeto de ML
- Explorar e analisar dados (EDA)
- Treinar múltiplos modelos de classificação
- Comparar algoritmos
- Selecionar o melhor modelo
- Validar resultados com dados não vistos

## Dataset
- 150 amostras
- 4 atributos numéricos
- 3 classes (setosa, versicolor, virginica)

## Tecnologias
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn

## Modelos Utilizados
- Logistic Regression
- LDA
- KNN
- Decision Tree
- Naive Bayes
- SVM

## Avaliação
- Validação cruzada (k-fold = 10)
- Métricas: accuracy, precision, recall, f1-score
